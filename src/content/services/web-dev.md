## Wake Lock Engine

AwakeMode uses the Screen Wake Lock API to keep sessions active while your tab stays open and visible.

### Included

- Start and stop keep-awake sessions quickly.
- Handle supported, blocked, and unavailable wake lock states.
- Auto-reacquire wake lock after tab visibility changes.
- Keep controls simple for daily use.
