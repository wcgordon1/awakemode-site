Support and ops teams often run long data jobs from web dashboards.

## Goals

1. Prevent machine sleep while transfer jobs run.
2. Keep usage simple for non-technical staff.
3. Use timers so sessions end automatically.

## Result

Transfers finish with fewer interruptions and less operator supervision.
