QA teams run long browser automation suites that should not pause mid-run.

## Goals

1. Keep machines awake during CI-adjacent local runs.
2. Use session presets for recurring test windows.
3. Provide clear active/inactive wake status.

## Result

AwakeMode lowered interrupted runs and improved confidence for unattended test sessions.
